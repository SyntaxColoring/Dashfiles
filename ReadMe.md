Max's Dotfiles
==============
The way I like my Unixy things.
Comments, questions, suggestions and pull requests are welcome.

Installation
============
With [GNU Stow](https://www.gnu.org/software/stow):

1. Extract or clone this repo into `~/Dotfiles` (or similar).
2. `cd ~/Dotfiles`
3. `stow */`
